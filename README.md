# How to Run

1.  **Ensure MongoDB is running**:
    ```bash
    mongod
    ```
    (Or ensure the service is started on `mongodb://localhost:27017`)

2.  **Run the Bot**:
    ```bash
    python main.py
    ```

3.  **Access the Web Interface**:
    Open [http://localhost:8000](http://localhost:8000) in your browser.

4.  **Login**:
    Use your API ID and Hash.
